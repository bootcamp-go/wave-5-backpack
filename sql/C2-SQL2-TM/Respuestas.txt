¿A qué se denomina JOIN en una base de datos?
Se denomina JOIN a combinar datos de una tabla con los datos de otra u otras.

Nombre y explique 2 tipos de JOIN.
INNER JOIN: Combinacion de datos siempre que encuentre una coincidencia en los campos que le indiquemos
LEFT JOIN: Trae todos los datos de la tabla principal (izquierda) y los registros que coinciden de la tabla secundaria (derecha)

¿Para qué se utiliza el GROUP BY?
GROUP BY se utiliza para agrupar datos segun las columnas indicadas

¿Para qué se utiliza el HAVING?
HAVING se utiliza para filtrar los datos agrupados con GROUP BY
