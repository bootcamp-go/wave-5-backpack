1. ¿A qué se denomina JOIN en una base de datos?
es una union o realcion de datos de dos tablas de una base de datos
2. Nombre y explique 2 tipos de JOIN.
INNER JOIN : es la interseccion de datos de dos tablas , generalmente se relaciona una primary key
con una foreign key para obtener mas informacion como conjunto de datos 
LEFT JOIN : es la union dela tabla izquierda con las coincidencia o interseccion de la tabla derecha (tabla o conjunto)
